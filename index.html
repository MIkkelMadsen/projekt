PHP ^

PHP er et programmeringssprog brugt til alt, men det er mest brugt til hjemmesider. Lavet af Rasmus Lerdorf, udgivet I 1995. PHP er et meget stabilt sytem som er brugt på 78.8% af alle hjemmeside ifølge W3Techs

C++ ^

C++ er et programmeringssprog brugt til alt, men det er brugt meget i Game Hacking(Snyd i Spil). Lavet af Bjarne Stroustrup, udgivet I 1985. C++ er brugt af Google, YouTube, Facebook, Yahoo, Amazon, Twitter, Bing. C++ er også brugt i programmer som Firefox, Google Programmer, og Apple OS X

C# ^

C# er et programmeringssprog primært brugt til at lave applikationer/programmer. Lavet af Anders Hejlsberg, udgivet I 2000. C# blevet brugt til Photoshop, Microsoft Office, Skype, Paint.NET, Internet Explorer, og selv noget af Facebook er lavet i C#

Java ^

Java er et programmeringssprog brugt til alt. Lavet af James Gosling, udgivet I 1995. Java er brugt til at lave mange forskelige ting, det er mest kendt for spillet Minecraft. Men det er også brugt til mange forskelige hjemmesider som Google, YouTube, Facebook, Yahoo, Amazon, Twitter, eBay, og Linkedin

Javascript ^

Javascript er et programmeringssprog til browsere, lavet af Brendan Eich i Mid til Slut 1995, med hjælp fra Ecma International, en IT organisation. Javascript er brugt til mange/fleste hjemmesider, inkl. denne.

Lua ^

Lua er et programmeringssprog primært brugt til at lave ekstra indhold til applikationer/programmer. Lavet af Roberto Ierusalimschy, udgivet I 1993. Et eksempel er spillet Garry's Mod (Forkortet til GMod), lavet af Garry Newman. Der er Lua brugt til at lave forskellige tilføjelser.

Python ^

Python er et programmeringssprog brugt til alt. Lavet af Guido Van Rossum, udgivet I 1991. Python er brugt af Wikipedia, Google, Yahoo!, NASA, Facebook, Amazon, Instagram, Spotify, og hele Reddit er lavet I Python.
